终于搭好我的个人博客啦！

🎉 终于搭好我的个人博客啦！

折腾了好几天，踩了无数坑，我的个人博客终于正式上线了！

这篇文章就当是一个小小的纪念，也顺便记录一下建站过程里的心得。

为什么要搭博客？

说起来，搭博客的想法其实很早就有了，主要是这几个原因：

  • ✅ 记录学习和生活的碎片
  • ✅ 整理自己的笔记和心得，方便以后回顾
  • ✅ 给喜欢的东西一个属于自己的小角落

这次的配置

这次用的是：

  • Hexo 静态博客框架
  • Magnetic 主题（清新简约风格）
  • 托管在 GitHub Pages，绑定了自定义域名 ppgames.sbs

虽然过程里遇到了不少问题，比如域名缓存、部署不同步之类的，但现在终于都解决了！

以后想在这里写点什么？

  1. 学习笔记：专业相关的内容，还有各种踩坑记录
  2. 日常分享：看到的有趣东西、生活里的小发现
  3. 偶尔的碎碎念，想到什么就写点什么

最后

希望这个小博客能一直写下去，也欢迎大家常来逛逛～

以后的文章，都会在这里更新！
